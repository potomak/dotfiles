# dotfiles

They include minimal configurations for:

 * vim
 * tmux

## Usage

Run:

```sh
$ cd
$ mkdir src
$ cd src
$ git clone git@github.com:potomak/dotfiles.git
$ cd
$ ln -s ~/dotfiles/.vimrc .vimrc
$ ln -s ~/dotfiles/.tmux.conf .tmux.conf
```
